Список определений

SGML
Standard Generalized Markup Language - стандартизованный обобщенный язык разметки.SGML является прародителем большинства современных
языков разметки.
HTML
HyperText Markup Language - язык разметки гипертекста. В последнее время XHTML пользуется огромной популярностью, благодаря стремительному темпу роста доступности мировой сети - Internet.
CSS
Cascading Style Sheets - каскадные таблицы стилей. Используются для визуального оформления элементов XHTML.
XML
eXtensible Markup Language - расширяемый язык разметки. C его помощью можно создавать собственный язык разметки, создавая словарь элементов и определяя их внешний вид в браузере.
